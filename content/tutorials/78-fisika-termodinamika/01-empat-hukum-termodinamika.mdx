# Termodinamika: Ilmu yang Menguasai Energi Alam Semesta

Termodinamika lahir dari masalah praktis revolusi industri: *Seberapa efisien sebuah mesin uap bisa bekerja?* Sadi Carnot pada 1824 menemukan jawaban yang mengejutkan вҖ” ada batas efisiensi yang tidak bisa ditembus, bukan karena keterbatasan rekayasa, tapi karena **hukum alam yang fundamental**.

Hari ini, termodinamika menjelaskan mengapa waktu hanya mengalir ke satu arah, mengapa kita tidak bisa membuat mesin yang bekerja selamanya, dan bahkan mengapa alam semesta itu sendiri pasti berakhir.

---

## 1. Konsep Dasar Termodinamika

```
Sistem: bagian yang kita pelajari
Lingkungan: segalanya di luar sistem
Batas: pemisah sistem dan lingkungan

Jenis Sistem:
  Terbuka:    massa DAN energi bisa bertukar dengan lingkungan
  Tertutup:   hanya ENERGI yang bertukar (tidak ada massa masuk/keluar)
  Terisolasi: tidak ada pertukaran sama sekali (О”E=0, О”m=0)

Variabel Keadaan (State Variables):
  Menyatakan keadaan sistem: T, P, V, n, U, H, S, G
  Tidak bergantung pada sejarah sistem вҖ” hanya keadaan sekarang

Variabel Jalur (Path Variables):
  q (kalor) dan w (kerja): bergantung cara proses berlangsung
  Оҙq вү  dU, harus ditulis sebagai proses, bukan beda keadaan
```

## 2. Hukum Nol Termodinamika: Dasar Termometer

```
"Jika A dalam kesetimbangan termal dengan C, dan B dalam
kesetimbangan termal dengan C, maka A dalam kesetimbangan
termal dengan B."

A ~ C, B ~ C вҶ’ A ~ B

Ini memungkinkan definisi SUHU dan penggunaan termometer.
Termometer = sistem C yang dijadikan standar.

Skala Suhu:
  Celsius (В°C):     0В°C = titik beku air, 100В°C = titik didih air
  Fahrenheit (В°F):  T_F = (9/5)T_C + 32
  Kelvin (K):       T_K = T_C + 273.15  вҶҗ skala absolut, 0K = nol absolut
  Rankine (В°R):     T_R = T_F + 459.67  вҶҗ Fahrenheit absolut

Nol Absolut (0 K = -273.15В°C):
  Suhu terendah yang bisa dicapai secara teoritis
  Semua gerakan termal berhenti
  Tidak bisa dicapai sempurna (Hukum Ketiga)
```

## 3. Hukum Pertama: Kekekalan Energi

```
О”U = q + w     (konvensi IUPAC)
О”U = q - w     (konvensi teknik lama)

О”U = perubahan energi dalam (internal energy)
q   = kalor yang diserap sistem dari lingkungan
      q > 0: sistem menyerap panas (endotermik dari perspektif sistem)
      q < 0: sistem melepas panas (eksotermik)
w   = kerja yang dilakukan PADA sistem
      w > 0: lingkungan melakukan kerja pada sistem (kompresi)
      w > 0: sistem melakukan kerja pada lingkungan (ekspansi) вҶ’ w negatif

Kerja P-V (ekspansi/kompresi gas):
  w = -вҲ«P_ext dV
  
  Untuk proses kuasistatik (sangat lambat):
  w = -вҲ«P dV   (menggunakan P_dalam, hampir sama dengan P_ext)

Proses Umum:
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ Proses       в”Ӯ Hubungan                                  в”Ӯ
в”ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө
в”Ӯ Isokhorik    в”Ӯ V konstan вҶ’ w=0 вҶ’ О”U = q_v               в”Ӯ
в”Ӯ Isobarik     в”Ӯ P konstan вҶ’ w=-PО”V вҶ’ О”U = q_p - PО”V     в”Ӯ
в”Ӯ Isotermal    в”Ӯ T konstan вҶ’ О”U=0 (gas ideal) вҶ’ q=-w       в”Ӯ

```

### Entalpi (H):
```
H = U + PV    [Joule]

О”H = О”U + О”(PV)
О”H = О”U + PО”V   (untuk P konstan)
О”H = q_p        (kalor pada tekanan konstan = perubahan entalpi)

вҶ’ Mengapa H penting?
  Reaksi kimia biasanya berlangsung di tekanan konstan (terbuka ke atmosfer)
  вҶ’ q_p = О”H lebih mudah diukur dan diinterpretasi

О”H < 0: reaksi EKSOTERMIK (melepas panas, sistem kehilangan energi)
О”H > 0: reaksi ENDOTERMIK (menyerap panas вҖ” butuh dipanaskan)
```

### Kapasitas Panas:
```
C_v = (вҲӮU/вҲӮT)_V    kapasitas panas pada volume konstan
C_p = (вҲӮH/вҲӮT)_P    kapasitas panas pada tekanan konstan

Untuk gas ideal monoatomik: C_v = (3/2)R, C_p = (5/2)R
Untuk gas ideal diatomik:   C_v = (5/2)R, C_p = (7/2)R
Ratio: Оі = C_p/C_v (monoatomik=5/3, diatomik=7/5=1.4)

Hukum Dulong-Petit (padatan):
  C_v вүҲ 3R = 24.9 J/(molВ·K)  untuk logam kristal
  Valid hanya pada T tinggi (quantum corrections at low T)
```

## 4. Hukum Kedua: Entropi dan Arah Proses

> **Entropi alam semesta selalu meningkat atau tetap dalam setiap proses yang terjadi secara spontan.**

```
Definisi Entropi (Clausius):
  dS = Оҙq_rev / T    [J/K]

  S = besaran state function (bergantung keadaan, bukan jalur)
  q_rev = kalor yang ditransfer secara reversibel

Ketidaksamaan Clausius:
  dS вүҘ Оҙq/T    (= untuk proses reversibel, > untuk proses irreversibel)

  О”S_total = О”S_sistem + О”S_lingkungan вүҘ 0
```

### Entropi sebagai Disorder (Boltzmann):

```
S = k_B Г— ln(W)     (persamaan Boltzmann)

k_B = 1.381 Г— 10вҒ»ВІВі J/K
W = jumlah mikrokonfigurasi yang mungkin (multiplicity)

W besar вҶ’ S besar вҶ’ lebih "acak/disorder"

Contoh intuisi:
  Gas di satu sisi kotak вҶ’ W kecil, S rendah
  Gas menyebar ke seluruh kotak вҶ’ W sangat besar, S tinggi
  вҶ’ Gas SPONTAN menyebar! (meningkatkan S_total)

Mengapa telur tidak bisa "tidak pecah"?:
  Telur pecah: W sangat besar (banyak cara susunan keping)
  Telur utuh: W kecil (satu konfigurasi khusus)
  P(kembali utuh) = W_utuh/W_pecah вүҲ 10^(-Milyaran) вүҲ 0
```

### Perubahan Entropi dalam Berbagai Proses:

```
Ekspansi Isotermal Gas Ideal (n mol):
  О”S = nR ln(VвӮӮ/VвӮҒ) = nR ln(PвӮҒ/PвӮӮ)

Pemanasan pada P konstan:
  О”S = nC_p ln(TвӮӮ/TвӮҒ)

Perubahan fasa pada T konstan:
  О”S_leleh  = О”H_leleh / T_leleh
  О”S_didih  = О”H_didih / T_didih

  HвӮӮO mendidih (100В°C = 373 K, О”H_vap = 40.7 kJ/mol):
  О”S = 40700/373 = 109.1 J/(molВ·K)
  (besar! вҶ’ gas jauh lebih disordered dari cairan)

Aturan Trouton: О”S_vap вүҲ 88 J/(molВ·K) untuk sebagian besar cairan
  (ikatan hidrogen menyebabkan nilai lebih besar: air=109, HF=113)
```

## 5. Mesin Carnot dan Efisiensi Maksimum

```
Mesin panas (heat engine): menyerap panas, menghasilkan kerja

        Q_H (dari reservoir panas T_H)
         вҶ“
    [MESIN] вҶ’ W_kerja
         вҶ“
        Q_C (dibuang ke reservoir dingin T_C)

W = Q_H - Q_C    (Hukum Pertama)
Efisiensi: О· = W/Q_H = 1 - Q_C/Q_H

Mesin Carnot (siklus ideal dan reversibel):
  О·_Carnot = 1 - T_C/T_H    (dalam Kelvin!)

  T_H = suhu reservoir panas
  T_C = suhu reservoir dingin

  Ini adalah BATAS ATAS efisiensi вҖ” tidak ada mesin nyata
  yang bisa melebihi efisiensi Carnot!

Contoh PLTU Batubara (T_H=600В°C=873K, T_C=30В°C=303K):
  О·_Carnot = 1 - 303/873 = 0.653 = 65.3% (teoritis)
  О·_nyata вүҲ 35-45% (karena irreversibilitas, gesekan, dll)
```

### Pernyataan Hukum Kedua:

```
Kelvin-Planck: Tidak mungkin membuat mesin yang beroperasi dalam
siklius dan mengubah kalor sepenuhnya menjadi kerja.

Clausius: Tidak mungkin memindahkan kalor secara spontan dari
benda dingin ke benda panas tanpa kerja luar.

Kedua pernyataan ini EKUIVALEN (terbukti satu menyiratkan yang lain).
вҶ’ Broken вҶ’ perpetual motion machine of the second kind impossible
```

## 6. Hukum Ketiga: Nol Absolut

```
"Entropi zat kristal murni sempurna adalah nol pada suhu 0 Kelvin."
S(0 K) = 0     (konfigurasi tunggal, W=1, ln1=0)

Konsekuensi:
  1. 0 K tidak bisa dicapai dalam jumlah langkah terbatas
     (setiap langkah pendinginan memberikan hasil yang berkurang)
  2. Memberikan referensi mutlak untuk entropi (berbeda dengan U dan H)
  3. Entropi ABSOLUT dapat dihitung: SВ°(298K) = вҲ«вӮҖВІвҒ№вҒё (C_p/T)dT

Rekord suhu terendah yang dicapai:
  ~10вҒ»В№вҒ° K (100 picokelvin) вҖ” Bose-Einstein Condensate di lab MIT
  Bintang Neutron: ~10вҒ¶ K (jauh lebih panas!)
  0 K sendiri: tidak pernah dicapai
```

## 7. Siklus Termodinamik di Mesin-Mesin Nyata

```
Siklus Otto (Mesin Bensin):
  1вҶ’2: Kompresi Adiabatik
  2вҶ’3: Pembakaran (V konstan, isokhorik)
  3вҶ’4: Ekspansi Adiabatik
  4вҶ’1: Pembuangan (V konstan)
  
  О·_Otto = 1 - (1/r^(Оі-1))   r = rasio kompresi (~8-12 untuk bensin)
  r=10, Оі=1.4: О· = 1 - 10^(-0.4) = 60.2% (teoritis)
  Aktual: ~25-35%

Siklus Diesel:
  Kompresi lebih tinggi (r=14-25) вҶ’ efisiensi lebih baik (~40-50% aktual)
  Pembakaran pada P konstan (isobarik)

Siklus Rankine (Pembangkit Uap):
  1вҶ’2: Pompa (kompresi cairan)
  2вҶ’3: Boiler (absorb panas, uap bertekanan)
  3вҶ’4: Turbin (ekspansi, kerja)
  4вҶ’1: Kondenser (buang panas)

Siklus Brayton (Turbin Gas, Mesin Jet):
  1вҶ’2: Kompresor
  2вҶ’3: Ruang Bakar
  3вҶ’4: Turbin   вҶҗ menghasilkan kerja
  4вҶ’1: Buang ke atmosfer
```
