# Teknik Integrasi: Seni Menemukan Antiturunan

Jika diferensial adalah tentang **mengurai perubahan sekecil mungkin**, maka integral adalah tentang **menumpuk serpihan-serpihan kecil itu kembali menjadi keseluruhan**. Tapi tidak setiap integral bisa diselesaikan dengan formula dasar — diperlukan teknik-teknik khusus, seperti seorang kunci yang membutuhkan kuncinya yang tepat.

Di bab ini kita akan menguasai toolkit integrasi yang digunakan oleh insinyur, fisikawan, dan matematikawan di seluruh dunia.

---

## 1. Rekap: Integral Dasar yang Wajib Hafal

```
∫ xⁿ dx        = xⁿ⁺¹/(n+1) + C          (n ≠ -1)
∫ 1/x dx       = ln|x| + C
∫ eˣ dx        = eˣ + C
∫ aˣ dx        = aˣ/ln(a) + C
∫ sin(x) dx    = -cos(x) + C
∫ cos(x) dx    =  sin(x) + C
∫ sec²(x) dx   =  tan(x) + C
∫ 1/√(1-x²)dx =  arcsin(x) + C
∫ 1/(1+x²) dx =  arctan(x) + C
```

## 2. Teknik Substitusi (u-substitution)

Substitusi digunakan saat integran mengandung fungsi komposit f(g(x)) × g'(x):

```
Langkah:
1. Pilih u = g(x) (biasanya fungsi "dalam")
2. Hitung du = g'(x) dx
3. Ganti semua x dengan u
4. Integrasikan terhadap u
5. Substitusi balik u → g(x)
```

**Contoh 1:** `∫ 2x × (x² + 1)⁵ dx`
```
u = x² + 1    →    du = 2x dx

∫ (x² + 1)⁵ × 2x dx = ∫ u⁵ du
                     = u⁶/6 + C
                     = (x² + 1)⁶/6 + C   ✓
```

**Contoh 2:** `∫ cos(3x) dx`
```
u = 3x    →    du = 3 dx    →    dx = du/3

∫ cos(u) × du/3 = (1/3)∫ cos(u) du
               = (1/3) sin(u) + C
               = (1/3) sin(3x) + C   ✓
```

**Contoh 3 (sedikit lebih rumit):** `∫ x/√(x²-4) dx`
```
u = x² - 4    →    du = 2x dx    →    x dx = du/2

∫ x/√(x²-4) dx = (1/2)∫ u^(-1/2) du
               = (1/2) × 2u^(1/2) + C
               = √(x²-4) + C   ✓
```

## 3. Integrasi Parsial (Integration by Parts)

Untuk integran berupa **perkalian dua fungsi** yang berbeda "tipe":

```
∫ u dv = uv - ∫ v du

Panduan memilih u (LIATE):
  L - Logaritma  (ln, log)
  I - Invers trig (arcsin, arctan)
  A - Aljabar    (xⁿ, polinomial)
  T - Trig       (sin, cos, tan)
  E - Eksponensial (eˣ, aˣ)

Pilih u = fungsi yang tipe-nya LEBIH KIRI dalam LIATE
```

**Contoh 1:** `∫ x eˣ dx`
```
u = x         →  du = dx
dv = eˣ dx    →  v  = eˣ

∫ x eˣ dx = x×eˣ - ∫ eˣ dx
          = x eˣ - eˣ + C
          = eˣ(x - 1) + C   ✓
```

**Contoh 2:** `∫ x² ln(x) dx`
```
u = ln(x)    →  du = 1/x dx
dv = x² dx   →  v  = x³/3

∫ x² ln(x) dx = (x³/3)ln(x) - ∫ (x³/3)(1/x) dx
              = (x³/3)ln(x) - (1/3)∫ x² dx
              = (x³/3)ln(x) - (1/3)(x³/3) + C
              = (x³/3)ln(x) - x³/9 + C   ✓
```

**Contoh 3 (integrasi parsial dua kali):** `∫ x² sin(x) dx`
```
Putaran 1:
u = x²      →  du = 2x dx
dv = sin(x) →  v  = -cos(x)

= -x² cos(x) + 2∫ x cos(x) dx

Putaran 2 untuk ∫ x cos(x) dx:
u = x       →  du = dx
dv = cos(x) →  v  = sin(x)

= x sin(x) - ∫ sin(x) dx = x sin(x) + cos(x)

Gabungan:
∫ x² sin(x) dx = -x² cos(x) + 2[x sin(x) + cos(x)] + C
               = -x² cos(x) + 2x sin(x) + 2cos(x) + C   ✓
```

## 4. Integral Parsial Pecahan (Partial Fractions)

Untuk integran berupa **pecahan rasional** P(x)/Q(x):

```
∫ (3x + 1) / [(x+1)(x-2)] dx

Dekomposisi partial fractions:
(3x + 1) / [(x+1)(x-2)] = A/(x+1) + B/(x-2)

Kalikan kedua ruas dengan (x+1)(x-2):
3x + 1 = A(x-2) + B(x+1)

Cari A: substitusi x = -1:
3(-1) + 1 = A(-1-2) + 0
-2 = -3A → A = 2/3

Cari B: substitusi x = 2:
3(2) + 1 = 0 + B(2+1)
7 = 3B → B = 7/3

∫ (3x+1)/[(x+1)(x-2)] dx = ∫ (2/3)/(x+1) dx + ∫ (7/3)/(x-2) dx
                          = (2/3)ln|x+1| + (7/3)ln|x-2| + C  ✓
```

## 5. Aplikasi Integral: Luas Daerah

**Luas daerah antara kurva f(x) dan sumbu X, dari a ke b:**
```
A = ∫ₐᵇ |f(x)| dx

Jika f(x) ≥ 0: A = ∫ₐᵇ f(x) dx
Jika f(x) ≤ 0: A = -∫ₐᵇ f(x) dx   (negatif karena di bawah sumbu X)
```

**Luas antara dua kurva f(x) dan g(x):**
```
A = ∫ₐᵇ |f(x) - g(x)| dx = ∫ₐᵇ [f(x) - g(x)] dx   jika f(x) ≥ g(x)
```

**Contoh:** Luas antara y = x² dan y = x, dari x=0 ke x=1:
```
Cek: x ≥ x² untuk 0 ≤ x ≤ 1  (karena x² ≤ x saat 0≤x≤1)

A = ∫₀¹ (x - x²) dx
  = [x²/2 - x³/3]₀¹
  = (1/2 - 1/3) - (0 - 0)
  = 3/6 - 2/6 = 1/6 satuan luas   ✓
```

## 6. Volume Benda Putar

**Metode Cakram (Disk Method)** — putar kurva terhadap sumbu X:
```
V = π ∫ₐᵇ [f(x)]² dx
```

**Metode Cincin (Washer Method)** — putar daerah antara dua kurva:
```
V = π ∫ₐᵇ {[f(x)]² - [g(x)]²} dx
```

**Metode Kelopak (Shell Method)** — putar terhadap sumbu Y:
```
V = 2π ∫ₐᵇ x × f(x) dx
```

**Contoh — Bola dengan jari-jari r:**
Putar setengah lingkaran `y = √(r² - x²)` terhadap sumbu X:
```
V = π ∫₋ᵣʳ [√(r² - x²)]² dx
  = π ∫₋ᵣʳ (r² - x²) dx
  = π [r²x - x³/3]₋ᵣʳ
  = π [(r³ - r³/3) - (-r³ + r³/3)]
  = π × (4r³/3)
  = (4/3)πr³   ← rumus volume bola yang kita kenal!   ✓
```

## 7. Integral Tak Wajar (Improper Integrals)

Integral dengan batas tak hingga atau integran tak terbatas:

```
∫₁^∞ 1/x² dx = lim(b→∞) ∫₁ᵇ x⁻² dx
             = lim(b→∞) [-1/x]₁ᵇ
             = lim(b→∞) (-1/b + 1)
             = 0 + 1 = 1   (konvergen!)

∫₁^∞ 1/x dx = lim(b→∞) [ln|x|]₁ᵇ
             = lim(b→∞) ln(b) - 0
             = ∞   (divergen!)
```

> [!NOTE] Paradoks Gabriel's Horn
> Putar kurva y=1/x (x≥1) terhadap sumbu X:
> - **Volume** = π × ∫₁^∞ (1/x²) dx = π × 1 = **π** (terbatas!)
> - **Luas permukaan** = 2π × ∫₁^∞ (1/x)√(1+1/x⁴) dx ≈ **∞** (tak terbatas!)
>
> "Terompet yang bisa diisi cat, tapi tidak bisa dicat!" — paradoks matematika yang nyata.

## 8. Tabel Teknik Integrasi: Memilih Strategi

| Bentuk Integran | Teknik |
|----------------|--------|
| f(g(x)) × g'(x) | Substitusi u = g(x) |
| Produk fungsi berbeda tipe | Integrasi Parsial (LIATE) |
| Pecahan rasional P(x)/Q(x) | Partial Fractions |
| √(a²-x²), √(a²+x²), √(x²-a²) | Substitusi Trigonometri |
| sinᵐ(x)cosⁿ(x) | Identitas trig + substitusi |
| Batas tak hingga | Integral Tak Wajar + limit |
