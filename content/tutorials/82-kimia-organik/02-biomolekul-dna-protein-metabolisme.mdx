# Biomolekul: Kimia yang Membuat Anda Hidup

Setiap detik, triliunan reaksi kimia berlangsung di dalam tubuh Anda — tanpa Anda sadari. Otot Anda berkontraksi karena protein myosin dan actin berinteraksi dengan ATP. Pikiran Anda terbentuk karena neuron melepaskan neurotransmiter. Gen Anda direplikasi karena DNA polymerase membaca dan menyalin kode DNA dengan akurasi luar biasa.

Semua ini adalah **kimia organik dalam aksi** — dan memahaminya adalah kunci untuk kedokteran, bioteknologi, farmasi, dan pemahaman kehidupan itu sendiri.

## 1. Karbohidrat: Bahan Bakar dan Arsitektur

```
Rumus umum: Cₙ(H₂O)ₙ atau CₙH₂ₙOₙ (mono- dan disakarida)

Klasifikasi:
  Monosakarida: 1 unit gula
    Triosa (3C): gliseraldehid, dihidroksiaseton (dalam glikolisis)
    Pentosa (5C): ribosa (RNA), deoksiribosa (DNA)
    Heksosa (6C): glukosa, fruktosa, galaktosa

  Disakarida: 2 monosakarida + ikatan glikosida
    Sukrosa = glukosa + fruktosa (gula meja)
    Laktosa = galaktosa + glukosa (gula susu)
    Maltosa  = glukosa + glukosa (gula malt)

  Polisakarida:
    Pati (amilosa + amilopektin): simpanan energi tumbuhan
    Glikogen: simpanan energi hewan (di hati dan otot)
    Selulosa: struktur dinding sel tumbuhan (ikatan β → tidak dicerna manusia)
    Kitin: eksoskeleton serangga, dinding sel fungi
```

### Struktur Glukosa:
```
Bentuk terbuka (Fischer):         Bentuk cincin (Haworth):
       CHO                              CH₂OH
       |                               /    \
HO-C-H     (Karbon 1=aldehida)        O      OH
       |                             |      |
  H - C - OH                         C      C
       |                              \    /
  HO - C - H                         (semua dalam cincin)
       |
  H - C - OH
       |
  H - C - OH
       |
    CH₂OH

α-D-Glukosa: OH di C1 ke bawah (aksial)
β-D-Glukosa: OH di C1 ke atas (ekuatorial) → selulosa (β-1,4 linkage)
```

### Metabolisme Karbohidrat:
```
Glikolisis (Sitoplasma):
  Glukosa + 2NAD⁺ + 2ADP + 2Pi → 2 Piruvat + 2NADH + 2ATP + 2H₂O

Siklus Krebs (Mitokondria):
  Piruvat → Asetil-CoA → Siklus asam sitrat
  → 3NADH + 1FADH₂ + 1GTP per siklus (per asetil-CoA)

Rantai Transpot Elektron:
  NADH, FADH₂ → ATP synthase → ATP
  NADH → 2.5 ATP, FADH₂ → 1.5 ATP

Total dari 1 glukosa: ~30-32 ATP
(Teoritis = 38 ATP, aktual lebih rendah karena kebocoran membran)
```

## 2. Lipid: Energi, Membran, dan Sinyal

```
Lipid = molekul hidrofobik/amfipatik yang larut dalam pelarut non-polar

Asam Lemak:
  Jenuh: tidak ada ikatan rangkap (padat pada suhu kamar)
    Palmitat: CH₃(CH₂)₁₄COOH  (C16:0)
    Stearat:  CH₃(CH₂)₁₆COOH  (C18:0)
  
  Tak jenuh tunggal (MUFA): 1 ikatan rangkap
    Oleat: C18:1 (Δ9) → dalam minyak zaitun, avokad
  
  Tak jenuh ganda (PUFA): 2+ ikatan rangkap
    Linoleat: C18:2 (Δ9,12) → omega-6 (esensial!)
    α-Linolenat: C18:3 (Δ9,12,15) → omega-3 (esensial!)
    DHA: C22:6 → omega-3 penting untuk otak

Trigliserida (Lemak Netral):
  Gliserol + 3 Asam Lemak (via ikatan ester)
  Simpanan energi → 9 kkal/gram (vs karbohidrat 4 kkal/g)

Fosfolipid (Membran):
  Gliserol + 2 Asam Lemak + Fosfat + Alkohol kepala
  AMFIPATIK: kepala polar (hidrofilik) + ekor nonpolar (hidrofobik)
  → Membentuk BILAYER spontan dalam air
  → Membran sel = dua lapis fosfolipid!
```

### Kolesterol dan Hormon Steroid:
```
Kolesterol C₂₇H₄₆O — 4 cincin steroid menyatu:
  Fungsi ganda:
  → Membran sel: mengatur fluiditas (menyisipi antara fosfolipid)
  → Prekursor hormon steroid (kortisol, aldosteron, testosteron, estrogen)
  → Prekursor vitamin D, asam empedu

LDL ("kolesterol jahat"): membawa kolesterol ke jaringan
HDL ("kolesterol baik"): membawa kolesterol kembali ke hati

Hormon Steroid:
  Testosteron: anabolik (massa otot), androgenisme
  Estrogen (estradiol): siklus reproduksi wanita, tulang
  Kortisol: respons stres, anti-inflamasi
  Aldosteron: keseimbangan Na⁺/K⁺ ginjal
```

## 3. Protein: Mesin Molecular Paling Serbaguna

### Asam Amino — Batu Bata Protein:
```
20 asam amino standar, semua dengan:
  NH₂     COOH
   |        |
   C ← R (rantai samping = penentu identitas)
   |
   H

Klasifikasi berdasarkan R-group:
  Non-polar (hidrofobik): Gly, Ala, Val, Leu, Ile, Pro, Phe, Trp, Met
  Polar tidak bermuatan: Ser, Thr, Cys, Tyr, Asn, Gln
  Bermuatan positif (+): Lys, Arg, His (basa)
  Bermuatan negatif (-): Asp, Glu (asam)

Ikatan peptida:
  -CO-NH- (antara COOH satu AA dan NH₂ AA berikutnya)
  Sifat: planar (parsial ikatan rangkap karena resonansi), tidak bisa rotasi

Polipeptida: rantai AA dihubungkan ikatan peptida
  N-terminus (NH₂ bebas) → ... AAₙ ... → C-terminus (COOH bebas)
```

### Empat Tingkat Struktur Protein:
```
Struktur Primer:
  Urutan AA (1D) — dikode oleh DNA
  Menentukan semua tingkat struktur di atasnya!

Struktur Sekunder:
  Motif lokal berulang karena ikatan H antara gugus -CO dan -NH

  α-Helix: rantai polipeptida berputar spiral, 3.6 AA/putaran
    Ikatan H: CO dari AA ke-n → NH dari AA ke-n+4
    Ekor side chain mengarah ke luar

  β-Sheet: rantai polipeptida berbentuk lembaran
    Parallel β-sheet: kedua rantai searah (N→C)
    Antiparallel β-sheet: rantai berlawanan arah (lebih stabil)

  Turns dan Loops: menghubungkan helices dan sheets

Struktur Tersier:
  Pelipatan 3D seluruh rantai polipeptida
  Distabilkan oleh:
    Ikatan disulfida (-S-S- dari 2 Cys)
    Ikatan hidrogen
    Interaksi ionik
    Interaksi hidrofobik (inti non-polar tersembunyi dari air)
    Van der Waals

Struktur Kuarterner:
  Kombinasi 2+ subunit polipeptida
  Hemoglobin: 2α + 2β subunit (tetramer)
  Kollagen: 3 helices melilit (triple helix)
```

### Catalysis Enzim:
```
Enzim = katalis biologis — protein (dan sebagian RNA)
  Menaikkan laju reaksi 10⁶ - 10¹² kali
  Tidak dikonsumsi — bisa digunakan berulang kali

Model Lock and Key:
  Substrat (S) persis cocok dengan Active Site enzim
  Analoginya: gembok dan kunci

Model Induced Fit:
  Active site BERUBAH BENTUK saat substrat mendekat
  Lebih akurat — kebanyakan enzim menggunakan ini

Kinetika Michaelis-Menten:
  E + S ⇌ ES → E + P
  
  v = V_max × [S] / (Km + [S])
  
  Km = konsentrasi substrat saat v = V_max/2
       (ukuran afinitas: Km kecil → afinitas tinggi)
  V_max = kecepatan maksimum saat semua enzim tersaturasi
  kcat = V_max/[E_total] = turnover number (berapa substrat/detik/enzim)

  Inhibitor:
    Kompetitif: bersaing dengan S di active site
               → Km naik, V_max tetap (bisa diatasi dengan [S] tinggi)
    Nonkompetitif: berikatan di tempat lain → mengubah konformasi
               → Km tetap, V_max turun (tidak bisa diatasi dengan [S] tinggi)
    Irreversibel: berikat permanen (racun, beberapa antibiotik)
```

## 4. Asam Nukleat: DNA dan RNA

### Struktur DNA:
```
Nukleotida = Fosfat + Gula (deoksiribosa) + Basa nitrogen

Empat basa:
  Purin (cincin ganda):    Adenin (A), Guanin (G)
  Pirimidin (cincin tunggal): Timin (T), Sitosin (C)

Pasangan basa (Watson-Crick):
  A = T  (2 ikatan hidrogen)
  G ≡ C  (3 ikatan hidrogen — lebih kuat!)

Struktur heliks ganda (Watson & Crick, 1953):
  Dua rantai antiparalel berpilin
  Gugus fosfat dan gula di luar (hidrofilik)
  Basa berpasangan di dalam (hidrofobik, stacking)
  Pitch = 3.4 nm per putaran (10 bp per putaran)
  Diameter = 2 nm
```

### Dogma Sentral Biologi Molekuler:
```
DNA → RNA → Protein

Replikasi DNA (DNA → DNA):
  DNA Polymerase: menyalin DNA template
  Berjalan 5'→3', semi-konservatif
  Proofreading: error rate ~10⁻⁹ per nukleotida

Transkripsi (DNA → mRNA):
  RNA Polymerase: membaca template DNA, membuat mRNA
  Promoter: sinyal "mulai" di DNA
  Terminator: sinyal "berhenti"
  Splicing (eukariot): intron dibuang, ekson disambung

Translasi (mRNA → Protein):
  Ribosom (rRNA + protein): membaca mRNA
  tRNA: membawa asam amino sesuai kodon
  Kodon: 3 nukleotida = 1 asam amino (64 kodon, 20 AA → degenerasi!)
  Start kodon: AUG (Met)
  Stop kodon: UAA, UAG, UGA (tidak ada AA)
```

### CRISPR-Cas9: Revolusi Genetika Modern:
```
CRISPR = Clustered Regularly Interspaced Short Palindromic Repeats

Komponen:
  Guide RNA (gRNA): mengarahkan Cas9 ke lokasi spesifik di DNA
  Cas9 protein: "gunting DNA" yang memotong heliks ganda

Mekanisme:
  1. gRNA dirancang untuk sesuai dengan urutan DNA target
  2. gRNA-Cas9 kompleks mencari urutan PAM (NGG untuk SpCas9)
  3. Cas9 memotong KEDUA rantai DNA
  4. Sel memperbaiki patahan:
     NHEJ (error-prone) → mutasi/knockout gen
     HDR (template) → penyisipan gen yang diinginkan

Aplikasi:
  Pengobatan penyakit: sickle cell, beta-thalassemia (kini disetujui FDA!)
  Ternak: babi non-alergenik, salmon tumbuh lebih cepat
  Tanaman: padi tahan kekeringan, tomat tahan busuk
  Diagnostik: SHERLOCK, DETECTR untuk COVID-19
  Kontroversi: "designer babies" — Jiankui He (2018, dihukum penjara China)
```

> [!IMPORTANT] Kimia Organik sebagai Pondasi Biologi
> Seluruh biologi molekuler adalah kimia organik yang beroperasi di lingkungan air pada suhu ~37°C. Memahami ikatan kovalen (peptida, glikosida, fosfodiester), ikatan hidrogen (dalam struktur DNA dan protein), interaksi hidrofobik (lipid bilayer), dan katalisis asam-basa (mekanisme enzim) adalah kunci memahami kehidupan pada tingkat fundamental.
