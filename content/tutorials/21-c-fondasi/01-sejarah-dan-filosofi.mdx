# Sejarah dan Filosofi C: Kebebasan Mutlak

Selamat datang di titik awal peradaban modern sistem komputer. Jika Anda membaca teks di layar ini, kemungkinan besar sistem operasi, browser web, hingga kernel dari perangkat Anda ditulis menggunakan keturunan dari bahasa yang akan kita pelajari: **C**.

Bahasa C tidak hanya sekadar bahasa pemrograman, melainkan sebuah pisau bedah yang memberikan akses penuh ke perangkat keras komputer.

## 1. Asal Usul: Dari B Menjadi C

Sejarah C bermula di awal 1970-an di Bell Labs, tempat di mana sistem operasi legendaris UNIX diciptakan.

Awalnya, sistem operasi ditulis dalam **Assembly**, bahasa tingkat rendah yang secara harfiah adalah instruksi mesin. Menulis OS dalam Assembly sangat melelahkan dan kodenya hanya bisa berjalan di jenis prosesor tertentu. Ketika ganti mesin, kodenya harus ditulis ulang dari nol.

Ken Thompson dan Dennis Ritchie membutuhkan bahasa tingkat tinggi ("high-level") yang memiliki performa setara Assembly, tetapi bisa dipindahkan antar mesin (portable).
Dari bahasa pendahulunya yang bernama **BCPL** (diciptakan Martin Richards), lahirlah bahasa **B** oleh Ken Thompson. Namun bahasa B terlalu lambat dan memiliki batasan yang tidak memadai untuk sistem operasi.

Kemudian, **Dennis Ritchie** mengembangkan versi B dengan menambahkan *Data Types* dan struktur memori yang kuat, sehingga lahirlah **Bahasa C** pada tahun 1972-1973.

UNIX, sistem operasi revolusioner tersebut, kemudian ditulis ulang menggunakan C. Sejak saat itu, supremasi C dimulai.

> [!CAUTION] Kebebasan Mutlak Ada Harganya!
> Filosofi utama C adalah: **"Percayai si programmer"**. C akan membiarkan Anda menimpa memori sembarangan, mencabut paksa alamat data, dan jika Anda keliru, komputer tidak akan memberi peringatan yang manis seperti di Python, program Anda akan hancur dan *crash* seketika (Segmentation Fault).

## 2. Kenapa Belajar C di Zaman Modern?

Bahasa pemrograman telah berkembang. Kita punya Python, JavaScript, Go, dan Rust. Mengapa masih belajar bahasa tua dari tahun 1970-an?

1. **Pemahaman Fundamental "Mental Logic"**: C memaksa Anda memahami bagaimana komputer benar-benar bekerja. Di Python (`x = 10`), Anda tidak peduli tentang letak RAM. Di C, Anda akan belajar tentang alamat *Memory Management*, register, dan siklus CPU. Anda akan berhenti menjadi programmer "penebak", dan menjadi "arsitek" sesungguhnya.
2. **Kecepatan Brutal**: Kode C dikompilasi menjadi bahasa mesin mentah. Tidak ada antarmuka mesin virtual di tengah (seperti Java JVM) atau interpreter lambat (seperti Python). Secara performa, C sering kali berada di garis finis duluan.
3. **Pondasi Sistem Dunia**: 
   - Linux Kernel ditulis dalam C. (Termasuk Android)
   - Windows Kernel (NT) sebagian besar ditulis dalam C.
   - Python Interpreter asli (CPython) ditulis dalam C.
   - Mikrokontroler (Arduino, alat medis, rem ABS mobil, mesin cuci) diprogram dengan C.

## 3. Garis Keturunan Bahasa C

Banyak bahasa populer mengadopsi sintaks dan paradigma pemikiran C, sering disebut "C-Family Languages". Karena ini, belajar C seperti belajar bahasa ibu. Jika Anda paham C, bahasa lain hanya terasa seperti dialek.

- **C++**: Anak sah dari C. C dengan tambahan Object-Oriented Programming (Class).
- **Go / Rust**: Cucu spiritual yang mengambil performa dari C, namun mengubah sistem keamanannya agar tidak mudah *crash* memori.
- **Java & C#**: Bahasa turunan yang lebih aman dan diurus oleh Garbage Collector. Sintaks utamanya persis (menggunakan kurung kurawal `{}`).
- **JavaScript & PHP**: Bahasa web yang mengambil penulisan sintaks kontrol (`if`, `for`, `while`) mirip C.

## Kesimpulan

Belajar C adalah ziarah yang harus dilakukan oleh programmer sejati. Ini adalah perjalanan meninggalkan zona nyaman *framework* moderen, menyelam turun ke kegelapan sirkuit memori, membangun senjata dari besi mentah, lalu naik ke permukaan sebagai programmer yang kebal terhadap eror-eror dangkal.
