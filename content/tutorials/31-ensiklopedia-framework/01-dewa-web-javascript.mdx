# JavaScript & TypeScript: Hegemoni Web Global

JavaScript dilahirkan dalam 10 hari pada tahun 1995 oleh Brendan Eich di Netscape. Awalnya, bahasa ini hanyalah sebuah mainan remeh untuk membuat tulisan berkedip di browser. Tidak ada yang menduga mainan ini kelak melahirkan "The JavaScript Framework Wars", era paling brutal nan produktif dalam industri *Front-End* di mana The Engineers memperebutkan kendali atas the Document Object Model (The DOM).

Mari membedah kerangka kerja agung yang menaklukan sejarah The JavasScript The Web.

## 1. The Front-End Trinity (Penakluk Browser)

### A. The Angular (Google) - The Enterprise Behemoth
- **Lahir Tahun**: 2010 (AngularJS), dirakit ulang total pada 2016 (Angular 2+).
- **Pembuat**: Google.
- **Tujuan Sejarah**: Sejak 2010, halaman web mulai rumit. Google lelah memanipulasi komponen the DOM secara telanjang bulat dengan the JQuery. Mereka menciptakan sebuah The Kerangka Raksasa yang super ketat (MVC Pattern). Jika Anda memakai the Angular, Anda dipaksa memakai The TypeScript dan cara kerja the Google murni.
- **Kelebihan**: Berstruktur Sangat ketat seperti Militer. Terbaik untuk aplikasi Bank dan Korporasi Raksasa. The Two-Way Data Binding ilahianya memanjakan pengembang.
- **Kekurangan**: Terlampau berat, kurva belajar bagaikan batu karang yang the menyiksa.

### B. The React (Facebook) - The Reactive Illusionist
- **Lahir Tahun**: 2013.
- **Pembuat**: Facebook (Jordan Walke).
- **Tujuan Sejarah**: Di the Facebook Ads, the DOM membesar dan sangat lambat jika the Angular The MVC di the gunakan. Sang the React menendang semua The Konsep MVC dan membawa sebuah The Ilusi Baru: **The Virtual DOM**. 
React bukanlah The Framework penuh, dia the "Library Murni". Ia membongkar The UI Component menjadi balok-balok the LEGO (The JSX). State (The Data) berubah = The React merender the Shadow Dummynya dulu, lalu mencari the beda mutlak, barulah ngehantam Layar Asli the DOM secara The Bedah Laser kilat.
- **Kehancuran The Kompetisi**: Secara mutlak the React mematikan the Lawan lawanya (The EmberJS, The Backbone) dengan the the Konsep Komponen Reactivenya, dan ia adalah sang RAJA DUNIA FRONT-END saat Ini!.

### C. The Vue.js - Sang Ksatria Penyendiri (The The Independent Elegance)
- **Lahir Tahun**: 2014.
- **Pembuat**: Evan You (Mantan Pegawai The Google Angular).
- **Tujuan Sejarah**: Evan the You merasa The Angular terlalu berat dan the React terlalu pusing "Membuat the Everything is JavaScript X (The JSX)". Dia menyatukan the The Kerapian the View-Model The Angular dan The Ringannya the Virtual DOM The React ke dalam bentuk *The Single File Component (.vue)* yang amat manis. The HTML Tetap HTML, The CSS The Scopeds Tetap CSS. 
- **The Fakta Mengerikan**: The Vue.JS didompleng MURNI Oleh THE KOMUNITAS the Open Source tanpa backed The Perusahaan Raksasa namun Kekuatanya berhasil Menguasai Seluruh Wilayah The Tiongkok and the Benua Asia sebagai the standard The Development!

- **Lahir Tahun**: 2016 (Rich Harris).
- **The Tujuan**: Mengutuk The ilusi The Virtual DOM The React dengan kata makian "Virtual DOM is Pure Overhead!".
